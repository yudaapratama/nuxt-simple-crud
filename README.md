# Nuxt Simple CRUD

Simple CRUD using Nuxt 3

## 💻 Tech Stack
- nuxt 3
- nuxt/ui
- nuxt/google-fonts
- pinia

## 🔨 Setup

Make sure to install the dependencies:

```bash
# npm
npm install
```

## 🏃 Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```
